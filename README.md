# EduDjango

## Project Overview
EduDjango is a comprehensive educational framework built using Django, aimed at enhancing the learning experience for students and educators alike. It serves as an interactive platform that hosts various learning resources, quizzes, and collaborative tools to facilitate education.

## Features
- User authentication and profile management
- Course creation and management
- Interactive quizzes and assessments
- Discussion forums for collaborative learning
- Responsive design for accessibility on all devices

## Technology Stack
- **Backend:** Django, PostgreSQL, Djangorestframework
- **Frontend:** HTML, CSS, JavaScript
- **Deployment:** Docker, AWS / Heroku for cloud hosting
